Introduo

O objetivo deste trabalho analisar a distribuio dos retornos do Ibovespa. Para o clculo dos retornos, utilizaremos o preo de fechamento ajustado dirio no perodo de 27 de abril de 2020 a 30 de julho de 2024. Os dados sero obtidos a partir do Yahoo Finance. Para acessar o banco de dados, clique aqui.

Neste estudo, ser plotado o histograma dos retornos dirios do Ibovespa para verificar graficamente qual distribuio de probabilidade melhor se ajusta aos dados. Com base na anlise grfica, ser possvel gerar valores aleatrios dos retornos para simulaes futuras. Alm disso, realizaremos trs testes de Kolmogorov-Smirnov para dados contnuos, com as seguintes hipteses nulas:

1. H0: Retorno segue uma distribuio Normal;
2. H0: Retorno segue uma distribuio T-studante com (n - 1) graus de liberdade;
3. H0: Retorno segue uma distribuio T-studante com (n - 2) graus de liberdade.

Metodologia

Os dados foram obtidos atravs da biblioteca yfinance, focando nos preos de fechamento ajustados do ndice Ibovespa. Aps a obteno dos dados, foram calculados os retornos dirios, que foram ento ordenados para posterior anlise.

Testes de Kolmogorov-Smirnov

Foram realizados trs testes de Kolmogorov-Smirnov para avaliar se os retornos dirios seguem uma distribuio Normal, uma distribuio T-studante com (n - 1) graus de liberdade e uma distribuio T-studante com (n - 2) graus de liberdade (gl). Estes testes permitem verificar a aderncia das distribuies tericas aos dados empricos.

Resultados Esperados

Atravs desta anlise, esperamos identificar a distribuio que melhor descreve os retornos do Ibovespa, o que poder auxiliar na modelagem de previses e na tomada de decises de investimento.

Para gerar a ECDF (Funo de Distribuio Acumulada) dos retornos reais, assumimos que a probabilidade de cada valor de ( \frac{1}{n} ). Assim, a probabilidade acumulada dada por:

  • $\text{ret}_1 = \frac{1}{n}$

  • $\text{ret}_2 = \frac{2}{n}$

  • $\text{ret}_n = 1$

Testes de Kolmogorov-Smirnov para Distribuio Normal

No cdigo abaixo, realizaremos o teste de Kolmogorov-Smirnov para verificar a probabilidade de que os valores reais se aproximam da distribuio normal.

In[]:
dados_norm = np.sort(np.random.normal(0, 1, n))
Fn_norm = norm.cdf(dados_norm)
In[]:
D1 = np.zeros(n)
D2 = np.zeros(n)
In[]:
# Preencher D_1 e D_2
for i in range(n):
    D1[i] = 1 / n - Fn_norm[i]
    D2[i] = Fn_norm[i] - (i - 1) / n

# Calcular D_calc
D_calc = np.max(np.maximum(D1, D2))
In[]:
# Simulao do p-valor
alpha = 0.05  # nvel de significncia
n_rep = 10000
In[]:
D1 = np.zeros(n)
D2 = np.zeros(n)
In[]:
contador = 0
for _ in range(n_rep):
    aux = np.sort(np.random.uniform(0, 1, n))
    for j in range(len(aux)):
        D1[j] = j/n - uniform.cdf(aux[j])
        D2[j] = uniform.cdf(aux[j]) - (j - 1) / n
    D = np.max(np.maximum(D1, D2))
    if D >= D_calc:
        contador += 1

pvalor = contador / n_rep

if pvalor < alpha:
    print("Rejeite H_0")
else:
    print("No rejeite H_0")

print(f'Estatstica do teste D: {D_calc}')
print(f'Valor-p: {pvalor}')

O teste de Kolmogorov-Smirnov foi aplicado para avaliar se os valores reais seguem uma distribuio normal. A estatstica do teste, $D$, 0.0293, que representa a maior diferena absoluta entre a funo de distribuio acumulada emprica (ECDF) dos dados reais e a funo de distribuio acumulada terica da normal.

O valor-p obtido 0.3253. Este valor-p indica a probabilidade de observar uma estatstica de teste to extrema quanto a observada, assumindo que a hiptese nula $H_0$ (de que os dados seguem uma distribuio normal) verdadeira.

Como o valor-p significativamente maior do que o nvel de significncia comum de 0.05, no rejeitamos a hiptese nula. Isso sugere que no h evidncias suficientes para concluir que os dados reais diferem da distribuio normal. Portanto, com base neste teste, os dados podem ser considerados como seguindo uma distribuio normal, de acordo com o critrio estabelecido.

Concluso

Com base neste estudo, podemos afirmar com 95% de confiana que os retornos dirios do ndice Ibovespa podem ser modelados utilizando a distribuio normal, ou as distribuies t de Student com $(n - 1)$ ou $(n - 2)$ graus de liberdade.

No entanto, para a gerao de valores aleatrios, optou-se por utilizar uma distribuio normal devido sua simplicidade na gerao da amostra. Em contraste, no setor financeiro, comum modelar dados de retornos com uma distribuio t de Student, pois esta distribuio melhor captura a ocorrncia de eventos extremos, como grandes variaes de preo em um nico dia, que so frequentes em dados financeiros.
